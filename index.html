Проблема

За 4 года студент:

  • Проходит десятки дисциплин
  • Разрабатывает множество продуктов
  • Изучает технологии из разных сфер разработки
  • Делает множественные выступления
Однако структура на протяжении времени ослабевает

Почему так происходит?

  • Не помнит названия того, что делал полгода назад
  • Не может рассказать о проекте за 2 минуты
  • Молчит там, где нужно думать вслух
  • Не знает, чего ожидать от рекрутера

Решение

Создать инструмент, который не учит заново - а помогает увидеть то, что уже знаешь

JuniorReady

веб-приложение для подготовки выпускника IT-специальности к первому собеседованию

  • Roadmap навыков
  • Чеклист soft skills
  • Тренажёр интервью
  • Карта курсов

Задача продукта

Разработать фреймворк, который будет удобно наполнять в зависимости от того опыта, который необходим выпускникам

Roadmap навыков

Студент выбирает трек: Frontend или Backend и отмечает технологии, которые уже знает

  • HTML
  • CSS
  • JS
  • React
  • Git
  • Django
  • Laravel
  • Express
  • ...

Впервые видит весь объём того, что освоил

Чеклист soft skills
